# kube-local (developing)
 -- What does it do now?  
 -- Almost configuring some useless stuff
### Set of scripts to make local kubernetes cluster using kvm/qemu 
Works on: ubuntu 24.04
## How to run:
### 1. Clone repo
```bash
git clone https://github.com/Tsuyakashi/kube-local.git
```
### 2. Run `./quickstart.sh`:
```bash
cd kube-local && ./quickstart.sh
```
